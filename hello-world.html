Hello World

Built in an afternoon with pelican, taking advantage of GitHub pages backing onto Fastly. Who has the time to set up a CDN?

Running a small pre-push hook courtesy of Getting Clojure, with some minor additions to ensure sensible messages each time:

#!/bin/bash
head="Update to latest"
body=`git log origin/master..HEAD --oneline`
make publish
cd output && git add -A && git commit -m "$head" -m "$body" && git push origin master

Getting Clojure: http://mavant.com/blog/2014/03/10/pelican-git-hooks-github-dot-io/
